== Description ==

Provides one-click theme updates for all Graph Paper Press themes. WARNING: This does not back up your theme. If you have modified the theme code in any way, your changes will be lost. You have been warned.

== Installation ==

1. Upload the whole `gpp-theme-updates` directory to the `/wp-content/plugins/` directory
1. Activate the 'GPP Theme Updates' plugin through the 'Plugins' menu in WordPress
1. Go to Dashboard -> Updates, select all themes that you want to update and click the update button.

== Frequently Asked Questions ==

1. Will this back up my existing themes? No.
1. Will I lose any customizations that I've made to the theme files themselves? Yes.
1. Will I lose any custom CSS that I've added to the Theme Options page? No.
